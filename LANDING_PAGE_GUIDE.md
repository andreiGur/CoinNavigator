## שלב 3: עיצוב עם Astra

### 3.1: Customize Theme

1. **WordPress Admin:**
   ```
   Appearance > Customize
   ```

2. **Header Settings:**
   - Logo: הוסף לוגו (אופציונלי)
   - Site Title: "CoinNavigator"
   - Tagline: "Real-Time Crypto Spread Comparison"

3. **Colors:**
   - Primary: #667eea (סגול)
   - Secondary: #764ba2 (סגול כהה)

4. **Typography:**
   - Headings: Bold, Modern Font
   - Body: Clean, Readable Font

---

### 3.2: Page Settings

1. **בעמוד שלך:**
   - Page Attributes > Template: "Full Width" (אם יש)
   - או: "Default Template"

2. **Page Settings:**
   - Hide Title: No (תן לכותרת להופיע)
   - Full Width: Yes (אם אפשר)

---

## שלב 4: הגדרת עמוד ראשי

### 4.1: Settings > Reading

1. **WordPress Admin:**
   ```
   Settings > Reading
   ```

2. **"Your homepage displays":**
   - בחר: "A static page"
   - Homepage: בחר את העמוד שיצרת
   - Posts page: (השאר ריק או בחר עמוד אחר)

3. **שמור שינויים**
